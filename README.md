# Metro Builder Game

## Overview
Metro Builder Game is a browser-based strategy game where players design a metro network by placing stations and connecting lines under simple rules and constraints.

The project focuses on interactive game logic, state management, and clean front-end structure using vanilla web technologies.

---

## Features
- Grid-based metro map
- Station placement and line connections
- Real-time validation of moves
- Simple, rule-driven gameplay

---

## Tech Stack
- HTML
- CSS
- JavaScript

---

## How to Run
Open `main.html` in a web browser.

---

- Front-end development fundamentals  
- Interactive UI and game logic in JavaScript
